Getting started

The tutorials and manuals available on this website use the R statistical programming environment. All you need to get started is the latest version of R, which you can download here. I also recommend using the R-Studio user interface for R, which you can download here. If you are new to R I recommend taking a short introductory course, although all tutorials include detailed step-by-step instructions. There are various free R courses for beginners available online (e.g., Getting started with R). R is the most powerful widely used statistical programming language and its popularity continues to increase—particularly among hydrologists. It's intuitive and easy to learn, so even if you've never programmed before you'll up and running in no time!
