Getting started

The tutorials and manuals available on this website use the R statistical programming environment. All you need to get started is the latest version of R. I also recommend using the popular R-Studio user interface for R. If you have never used R, I recommend taking one of the many free short introductory courses available online (e.g., Getting started with R). R is the most powerful and widely used statistical programming language, and its popularity continues to increase—particularly among hydrologists. It's intuitive and easy to learn, so even if you've never programmed before you'll up and running in no time!
