R tools for water resources analysis

The "reservoir" R package

Install the latest CRAN version of reservoir into your R library using:

> install.packages("reservoir")

Install the most up to date development version using:

> devtools::install.github("swd-turner/reservoir")

The official reservoir manual is in writing.

Other resources

Professor Riccardo Rigon's popular AboutHydrology blog features links to dozens of R resources for hydrologists.

Claudia Vitolo's blog features various useful tools, tips and tutorials for hydrologists using R. Also check out her R4Hydrology google group.
